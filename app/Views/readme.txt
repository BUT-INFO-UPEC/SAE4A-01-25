Les composants partagés dans toutes les pages sont stockés dans ce doccument pour eviter la redondance.

Les fichiers sont des balises html tel que le header, le nav, le footer et d'autres composants qui peuvent etres utilisés sur plusieurs pages.
Les balises doivent etres ouvertes et fermées dans le fichier de sorte que l'importation d'un fichier soit equivalent a l'importation d'une balise.
Les fichiers peuvent etre partiellement ou entièrement générés par des commandes php pour ajouter une dimmension d'automatisation au code, sans avoir a se soucier des differents parametres de cession (ex: menu d'utilisateur connécté ou bien menu de connexion pour un inviter dans le header).
